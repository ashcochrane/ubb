Design System v2 — Warm Stone + Terracotta · Apr 2026
Cormorant 700 · DM Sans · DM Mono
Replaces sage/olive v1 palette

Evolved from the original sage/olive design system. Retains the warm parchment foundation and DM Sans + Cormorant type pairing, but replaces the green accent palette with a warm stone + soft terracotta direction. The accent colour (#a16a4a) is used sparingly — active states, wordmark dot, focus rings. Data colours do the talking.

Logo & Identity

Same wordmark rules — Cormorant 700, lowercase. The dot colour changes to terracotta. Favicon background is terracotta with a dark charcoal dot.

shadcn-ui Token Mapping

Paste into globals.css under :root. Key change from v1: --primary is charcoal, --ring is terracotta.

:root {
  /* UBB v2 × shadcn-ui — warm stone + terracotta       */

  --background:           #f4f2ee;  /* bg-page            */
  --foreground:           #2e2520;  /* text-primary        */
  --card:                 #fcfaf7;  /* bg-surface          */
  --card-foreground:      #2e2520;
  --popover:              #fcfaf7;
  --popover-foreground:   #2e2520;
  --muted:                #ece8e2;  /* bg-subtle           */
  --muted-foreground:     #9a8e80;  /* text-muted          */

  /* Primary — dark charcoal (primary buttons)            */
  --primary:              #2e2520;
  --primary-foreground:   #fcfaf7;

  /* Secondary — accent ghost                             */
  --secondary:            #f8f0e8;  /* accent-ghost        */
  --secondary-foreground: #5a3520;  /* accent-text         */

  /* Accent — terracotta                                  */
  --accent:               #f0e0d2;  /* accent-light        */
  --accent-foreground:    #5a3520;

  /* Destructive                                          */
  --destructive:          #b84848;
  --destructive-foreground: #ffffff;

  /* Borders & inputs                                     */
  --border:               #e2ddd5;
  --input:                #e2ddd5;
  --ring:                 #a16a4a;  /* focus = terracotta  */

  --radius:               0.625rem; /* 10px                */
}
